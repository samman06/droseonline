## Introduction

**Drose Online** is a comprehensive Learning Management System (LMS) designed for Egyptian schools. It facilitates academic management, student tracking, assignment submission, attendance monitoring, and communication between teachers, students, and administrators.

---

## Getting Started

### Logging In

1. Navigate to the login page
2. Enter your email and password
3. Click "Sign In" or use the demo credentials dropdown

**Demo Credentials Available:**
- **Admin**: `admin@droseonline.com` / `admin123`
- **Teachers**: Various teacher accounts (see login page)
- **Students**: Various student accounts (see login page)

### First Time Login

After logging in for the first time:
1. Update your profile information
2. Change your password
3. Upload a profile picture (optional)
4. Review your dashboard

## ğŸ’¡ Tips & Best Practices

### For Teachers:
1. **Weekly Planning**: Create assignments at the start of the week
2. **Clear Instructions**: Provide detailed assignment descriptions
3. **Timely Grading**: Grade submissions within 48-72 hours
4. **Regular Attendance**: Take attendance at the start of each session
5. **Communication**: Use announcements for important updates

### For Students:
1. **Check Calendar Daily**: Review upcoming deadlines
2. **Submit Early**: Don't wait until the last minute
3. **Read Feedback**: Learn from teacher comments
4. **Ask Questions**: Contact teachers if unclear
5. **Track Progress**: Monitor your grades and attendance

### For Admins:
1. **Regular Backups**: Ensure data is backed up regularly
2. **User Management**: Keep user accounts up to date
3. **Monitor Activity**: Review system usage and performance
4. **Support Users**: Provide training and assistance
5. **Data Integrity**: Regularly audit data accuracy
